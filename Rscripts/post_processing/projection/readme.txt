All the result files are shapefiles and grid format (ASCII Grid) and they can be displayed in a GIS tool. 
If you have a problem, the files are defined in RGF93 - Lambert 93 projection, EPSG code 2154 (for metropolitan France).
The symbology of each file is available in .qml (Qgis) or .lyr (ArcGis) format.
For the results of submersions, it is recommended to display them with a discretization in 5 classes: 0; 0-0.5; 0.5-1; >1;
